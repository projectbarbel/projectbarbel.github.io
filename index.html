Bitemporal data management

BarbelHisto Core is a bitemporal in-memory data storage API. It keeps track of two time dimensions: effective time and record time.

In-memory transactions

BarbelHisto Core provides in-memory and persistent transaction isolation to support business critical applications.

Ultra-fast processing

Built on best-of-bread frameworks, BarbelHisto allows high-end performance batch processing operations.

Gets you started in minutes

Barbel core API can be used in various flavours. As full data storage API for microservices, as well as simple user-friendly API to manage your bitemporal data. All functions are provided by a simple API.

Built-in persistence support

Although it's not mandatory to operate with persistence storage backends, BarbelHisto provides support for off-heap, disk and custom persistence integration.
