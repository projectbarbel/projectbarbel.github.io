Bitemporal data

BarbelHisto Core is a Java framework for audit-proof bitemporal data management. It keeps track of two time dimensions: effective time and record time.

Simple API

The core functionality is provided in a very small intuitive API in order to simplify the bi-temporal data processing for the users.

Fast processing

Built on proven community frameworks, BarbelHisto can be configured for extremely fast processing scenarios.

Extension framework

Users can extend the functionality of BarbelHisto by using the configuration options and a small event-based extension framework.

Persistence support

BarbelHisto provides built-in transactional persistence support options and can be extended to integrate with other external data sources.

The goal of BarbelHisto is to enable developers to concentrate on the business requirements instead of implementing complex audit-proof bitemporal data storage again and again.
